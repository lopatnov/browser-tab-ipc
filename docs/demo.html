browser-tab-ipc

Real-time messaging between browser tabs

A lightweight TypeScript / JavaScript library for inter-process communication between browser tabs. Messages are delivered instantly using the best transport available in your browser — no server, no WebSockets, no configuration required.

How to use this demo

  1. Open this page in two or more browser tabs
  2. Select a transport and click Connect in each tab
  3. Type a message and press Enter or click Send
  4. The message appears instantly in all other connected tabs

Transport options

Auto Picks the best available transport automatically (recommended)
BroadcastChannel Fastest option — native browser API, same-origin only
SharedWorker Cross-origin capable — requires ipc-worker.js to be hosted locally
SessionStorage Universal fallback — works in every browser
